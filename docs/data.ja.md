# モデルデータ

!!! note "PSMpy v0.1 とは canonical テーブル経由で共有"
    本ページは参照 GAMS `PowerSystemModel` のデータ想定です。PSMpy は**同じ**データを
    検証済み canonical テーブル（モデルリポジトリの `docs/spec/data_schema.md`）として、
    GAMS 入力からの抽出またはデータパイプラインから取り込みます — したがって以下の想定は
    PSMpy にも当てはまります。

本ページでは各部門で使用したデータの想定を説明します。コスト・効率パラメータ、ポテンシャルの推計方法、既存設備の想定を部門ごとに記述します。
